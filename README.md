# Парсер матчей CS2

Проект на C++ для загрузки и просмотра расписания киберспортивных матчей через Pandascore API. 

Что реализовано:
- Свои структуры данных (сортированные таблицы, деревья) для работы с записями.
- Нормальный парсинг JSON. Если матч еще не начался и API выдает пустоту, прога не вылетает с ошибкой, а ставит TBD.
- Графический интерфейс на ImGui + OpenGL3.
- Данные скачиваются отдельным скриптом на питоне, чтобы не перегружать плюсовый код сетевыми запросами.

Как запустить:
1. Запустить python3 fetch.py и вставить токен
2. Собрать проект через make
3. Запустить ./db_engine
